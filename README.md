# Clash Airport Tools

包含 7 个命令：
- switch-clash
- add-airport
- remove-airport
- rename-airport
- update-airport-url
- show-airport-url
- update-airport

## 安装
```bash
sudo bash install.sh
```

## 卸载
```bash
sudo bash uninstall.sh
```

## 常用
```bash
switch-clash list
switch-clash edge
add-airport
remove-airport edge
rename-airport edge edge2
update-airport-url edge
show-airport-url edge

# 刷新指定机场的订阅节点
update-airport edge

# 刷新所有机场的订阅节点
update-airport all
```
